# Ícones da Aplicação Desktop

Para personalizar o ícone da aplicação desktop, coloque seus arquivos de ícone nesta pasta:

- `icon.ico` - Ícone para Windows (256x256 ou maior, formato .ico)
- `icon.png` - Ícone alternativo em PNG (256x256 ou maior)

## Como criar ícones:

### Usando ferramentas online:
1. Crie ou obtenha uma imagem 256x256 pixels
2. Converta para .ico em sites como: https://convertio.co/png-ico/
3. Salve como `icon.ico` nesta pasta

### Usando ferramentas locais:
- Windows: Use o GIMP ou Paint.NET
- Online: Use o favicon.io ou similar

## Ícone padrão:

Se nenhum ícone personalizado for fornecido, o Electron usará um ícone padrão.
